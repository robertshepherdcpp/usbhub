# house of wires
my fifth project for fallout!

**what is it?**
- the house of wires is a project to help remove the mess of wires! having a hub that usbs connect to creating a more organized layout
- the project is made to resemble a model house in which wires are connected and the project handles the rest (power etc.)
- the available usb options: 1 usb c power output, 3 usb a outputs (e.g. for charging your phone, charging your watch and anything else that needs charging)
- and then there is one usb input which spilts off into the other usb ports.

**how to use it?**
- print off the case and pcb (and purchase all items from the BOM)
- then to put the case on put the top half the case on the pcb (you know its the top if it has a door) and the base on the bottom, then using the 4M screws screw through. the parts of the screws pointing out the back
act as the base of the project!.
- then simply plug in a usb c port into the project (at the top)
- then connect the other usbs u need!
- just use these usbs as normal!

**how it works**
- the brain of this project is the SL2.1S, which splits the input usb into 5 channels!
- i didnt write any additional firmware for this project, that was all that was needed!

**how i made the project**
- made using kicad (for the pcb) and fusion (for the CAD design)
- using the fallout tutorial (as shown below) as a guide.

**my inspiration for the project/the why**
- i find that my desk is always messy with various wires, and i thought it would be cool to have a way to organise these (and do it in a fun way aswell!)
- i saw that there was a tutorial for a usb hub on [fallout](https://fallout.hackclub.com/docs/guided-projects/usb-hub), so i decided to give it a go (helping to improve my hardware skills!)
